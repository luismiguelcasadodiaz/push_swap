Detailed Description

arg_seen() check if num has been previously seen in the arguments.

Parameters
[in]numthe num to see if exists inside the stack.
[in]athe stack to save the arguments.
Returns
0 if the argument has not been seen previously. That is, it is not
inside the stack. 1 if the argument has been seen.


Loops the stack looking for num. If there is not, pushes num into the stack
and returns zero. Otherwiser returns one ==> arg_seen == True
